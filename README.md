# Open Graph Image Generator Assignment

## Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Dikshantw/OpenGraph.git
   cd opengraph
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the frontend:**

   ```bash
   npm run dev
   ```

4. **Run the server:**
   ```bash
   node server.js
   ```

Open http://localhost:3000 with your browser to see the result.
